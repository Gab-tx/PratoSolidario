
#  Queries Banco de dados Prato Solidário


>*Depois de construir e popular todo o seu banco de dados você deve desenvolver Queries simples para os problemas listados abaixo a fim de fazer buscas de dados:*

<br>

- [ ] Listar o nome e telefone de todos os doadores cadastrados.
- [ ] Exibir todos os alimentos com nome, tipo e valor.
- [ ] Listar o nome dos doadores e o id das doações que cada um realizou.
- [ ] Mostrar os alimentos que possuem data de validade posterior à data atual.
- [ ] Mostrar todas as entregas com data e status.
- [ ] Mostrar todas as doações com o nome da ONG que recebeu a doação.
- [ ] Exibir todas as entregas com o nome do voluntário responsável por cada uma.
- [ ] Mostrar todos os lotes de alimentação e a quantidade de alimentos em cada lote.
- [ ] Listar todos os alimentos pertencentes a um determinado lote de alimentação.
- [ ] Listar todos os voluntários com nome e email.
